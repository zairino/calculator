# Calculator

Simple calculator made to handle priority operations and decimal number

### Calculator tools
* PHP 8
* Symfony 5.2
* JQuery 3.6.0
* Twig 2.0

### Init project
* `docker-compose up -d --build`
* `docker exec -it php-fpm-calculator sh`
* `composer install`
* Go to `http://localhost:10006/`

### Testing
* `php bin/phpunit`
